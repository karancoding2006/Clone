# Amazon Clone — Project Progress

A front-end clone of Amazon's homepage built with HTML and CSS.

---

## ✅ What's Done

### Navbar
- Amazon logo (via external PNG URL)
- "Deliver To India" section with location icon
- Search bar with category dropdown, text input, and search icon
- Sign In / Accounts & Lists section
- Returns & Orders section
- Cart icon with label
- Hover border effect on all nav items

### Panel / Sub-navbar
- Hamburger menu ("All") with icon
- Navigation links: Today's Deals, Customer Services, Registry, Gift Cards, Sell
- "Shop deals in Electronics" promo text on the right

### Hero Section
- Background banner image (herobg.jpg)
- Amazon India redirect message bar with link

### Shop Section (Category Cards)
- 8 product category boxes in a responsive flex-wrap grid:
  - Get Your Game On
  - Electronics
  - Shop Fashion for Less (2×2 image grid)
  - Furniture
  - Smartphone
  - Home Decor
  - Backpacks
  - Snacks
- Each box has a title, product image, and "See more" link
